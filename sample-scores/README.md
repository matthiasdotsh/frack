# Sample scores

A few public domain orchestral parts to try frack with. All works are in
the public domain; the scans below are hosted on
[IMSLP](https://imslp.org/) and marked public domain there.

| `brahms-symphony-1-trombone-3-bass.pdf` | Brahms, *Symphony No. 1* Op. 68 – Trombone III (bass) | [IMSLP #967120](https://imslp.org/wiki/Special:ReverseLookup/967120) |

**Edition:** ed. Hans Gál (1890–1987), Leipzig: Breitkopf & Härtel,
n.d. (ca. 1927), plate Orch.B. 3204; reprinted New York: E. F. Kalmus,
n.d. [1933–70]. Part of the IMSLP Orchestra Parts Project.
**License:** public domain (Brahms † 1897; the edition/scan is public
